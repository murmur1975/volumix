# Volumix FAQ（よくある質問）

サービスローンチ後のサポート用FAQです。

---

## Q1: 処理後にファイルサイズがかなり変わっています

**A:** これは正常な動作です。

Volumix はラウドネス補正のために**音声ストリームを再エンコード**しています。そのため、元の音声のビットレートやエンコード方式によって、ファイルサイズが増減することがあります。

**ポイント:**
- **映像は無劣化**（コピー処理）のため、画質は変わりません
- 音声は AAC で再エンコードされます
- 元の音声が高ビットレートだった場合、サイズが減ることがあります
- 元の音声が低ビットレートだった場合、サイズが増えることがあります

ファイルサイズの変化が気になる場合は、処理前後の動画を再生して画質・音質に問題がないことをご確認ください。

## Q2: 処理結果が目標の ±1 LKFS に収まっていません

**A:** いくつかの原因が考えられます。

Volumix は最大5回の反復処理でターゲット値に近づけますが、コンテンツの特性によっては誤差が生じる場合があります。

**考えられる原因:**

1. **非常にダイナミックなコンテンツ**
   - 静寂と大音量が交互に繰り返されるような動画では、ラウドネス測定値が安定しにくいことがあります

2. **無音区間が多い**
   - 無音部分はラウドネス計算から除外されるため、無音の長さによって測定結果が変動することがあります

3. **True Peak リミッターの影響**
   - 音割れ（クリッピング）を防ぐため、ピークが 0 dBFS を超えないよう制限しています。これにより、目標値まで上げきれない場合があります

**対処法:**
- 結果が ±1.5 LKFS 以内であれば、一般的なプラットフォーム（YouTube、Podcastなど）の許容範囲内です
- より厳密な調整が必要な場合は、処理後のファイルを再度 Volumix に入力してください
