# Volumix リリース前 QAチェックリスト

リリース前に必ず実施する手動テスト項目です。
外部ツール（MediaInfoなど）を使用した品質確認も含みます。

## 1. 環境・準備
- [ ] クリーンインストール（古い設定/キャッシュが残っていないか）
- [ ] テスト用動画ファイルの準備
  - [ ] 短い動画（15秒程度）
  - [ ] 長い動画（5分以上）
  - [ ] ステレオ、モノラル混在
  - [ ] ラウドネスが低いファイル / 高いファイル

## 2. 機能制限（Free版 / Pro版）

### Free版（未認証状態）
- [ ] **バッジ表示**: 左上に「🆓 Free」が表示されている
- [ ] **同時処理制限**:
  - [ ] 複数ファイルをD&D → 「1ファイルまで」の警告が表示され、1つだけ追加される
  - [ ] リストに複数ある状態で処理開始 → 警告が表示され処理されない
  - [ ] 1ファイルのみ処理 → 成功する
- [ ] **設定制限**:
  - [ ] Sampling Rate が「Original」固定で変更不可
  - [ ] Bitrate が「VBR (自動)」固定で変更不可
- [ ] **レートリミット**:
  - [ ] 30分以内に11回目を処理しようとする → 警告が表示されブロックされる

### Pro版（`VOLUMIX-DEV-TEST-KEY` で認証）
- [ ] **ライセンス認証**: キー入力で認証成功し、「🌟 Pro」バッジに変わる
- [ ] **制限解除**:
  - [ ] 複数ファイルをD&D → すべて追加される
  - [ ] 複数ファイルを一括処理 → すべて成功する
  - [ ] Sampling Rate が変更可能
  - [ ] Bitrate が変更可能
  - [ ] 11回以上連続で処理してもブロックされない

## 3. 出力品質・仕様確認（MediaInfo / プレイヤー）

### 映像（Video）
- [ ] **再生確認**: 処理後ファイルが正常に再生できる（音ズレ、画面乱れなし）
- [ ] **画質**: 元ファイルと変化がない（`Video: copy` が効いているかMediaInfoで確認）

### 音声（Audio） - ラウドネス
- [ ] **-14 LKFS (Web)**:
  - [ ] 処理後、プレイヤーで聴感上の音量が適切か
  - [ ] (任意) Loudness Meter等で計測し、目標値の ±1.0 LUFS 程度に収まっているか
- [ ] **-24 LKFS (TV)**:
  - [ ] -14 LKFSより明らかに音が小さいか

### 音声（Audio） - フォーマット（Pro版機能）
**MediaInfo等で以下を確認**
- [ ] **サンプリングレート変更**:
  - [ ] 44.1kHz 指定 → Output: 44.1 kHz
  - [ ] 48kHz 指定 → Output: 48 kHz
  - [ ] 96kHz 指定 → Output: 96 kHz
- [ ] **ビットレート変更 (CBR)**:
  - [ ] 128k 指定 → Bit rate: Constant, 128 kb/s
  - [ ] 320k 指定 → Bit rate: Constant, 320 kb/s
- [ ] **VBR (Free/Default)**:
  - [ ] 指定なし → Bit rate mode: Variable (または記載なし)

## 4. UI/UX
- [ ] **進捗表示**: プログレスバーが正常に動く
- [ ] **キャンセル**: (もしあれば) 処理中にアプリを閉じてもゾンビプロセスが残らない
- [ ] **エラーハンドリング**:
  - [ ] 破損したファイルを読み込ませてアプリがクラッシュしないか

## 5. ライセンス管理
- [ ] **解除**: 「Deactivate」ボタンでFree版に戻る
- [ ] **再認証**: 解除後に再度キーを入れて認証できる

## 6. 多言語対応（i18n）
- [ ] **言語切り替え**: 設定モーダルで「日本語」「English」を切り替えられる
- [ ] **UI反映**: 切り替え後に全てのUI要素が正しく翻訳されている
- [ ] **購入リンク**: 日本語時は日本語版購入ページ、英語時は英語版購入ページに遷移する
